# Анализ продаж Superstore

## 📌 Описание проекта
Анализ датасета розничных продаж для выявления ключевых метрик и закономерностей.  
**Цель:** продемонстрировать навыки первичного анализа данных на Python.

## 📊 Использованные данные
- Датасет: Superstore Sales (Kaggle)
- Содержит информацию о продажах товаров, категориях, клиентах и географии

## 🔧 Инструменты
- Python (pandas, matplotlib)

## 🎯 Что сделано
1. Загружены и проверены данные (пропуски, дубликаты, аномалии)
2. Рассчитана общая выручка
3. Выявлены топ-категории товаров по выручке
4. Построена визуализация (столбчатая диаграмма продаж по категориям)

## 📈 Ключевые выводы
- Общая выручка составила **2.3 млн $**
- Самая прибыльная категория — **Office Supplies** (40% выручки)
- Категория **Technology** показала наибольшую среднюю цену чека
- Аномалий в ценах не обнаружено

## 🚀 Как запустить
Откройте файл `superstore_analysis.ipynb` в Jupyter Notebook или Kaggle/Colab
